Using the ATMega32U4 microprocessor to control various peripherials such as a LCD screen, user push-buttons, LEDs and an antenna.

1) ST7676R Dot Matrix LCD screen

Datasheet
https://www.lcd-module.de/eng/pdf/zubehoer/st7565r.pdf

This is a dot matrix LCD so I have paired it with a ASCII bitmap array in order to display text onto the LCD screen. See Bitmap below.

2) Push Buttons [4]

Mapped to PORTD for user control.

3) LEDs [8]

Mapped to PORTB for user control.

4) 2.4 GHz meander antenna

Not initiallized at the moment.
